# fsb-wasm
fsb-wasm is a WASM implementation of the FSB hashing algorithm, originally created by Daniel Augot, Matthieu Finiasz, Philippe Gaborit, St√©phane Manuel and Nicolas Sendrier. [Website](https://www.rocq.inria.fr/secret/CBCrypto/index.php?pg=fsb)

This code is based on the reference implementation, all rights reserved.

The JS wrapper for the library and the script `build.sh` is based on [crashdemons](https://github.com/crashdemons/)' work.

## Compiling

To compile this project, use `./build.sh`.
emscripten is required.
